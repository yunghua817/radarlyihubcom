01最佳化網速

讓 Codex 最佳化你的網速

能實現的效果

排查網路設定，整理可以嘗試的最佳化步驟。

複製給 Codex

請你作為網速最佳化助手，先檢查當前網路、DNS、代理、Wi-Fi、路由器和測速結果，給我一份從安全到激進的最佳化步驟；如果要改系統網路配置或重置設定，必須先解釋風險並等我確認。

02釋放空間

讓 Codex 給硬碟騰出空間

能實現的效果

找出可清理的快取、大檔案和下載殘留。

複製給 Codex

請你作為硬碟清理助手，先只讀檢查我的磁碟空間佔用，列出可以釋放空間的快取、大檔案、下載殘留和構建產物；不要直接刪除任何檔案，每一項都說明能釋放多少、風險是什麼，並等我確認。

X 實測 Kanoi 審批機制
03外掛修復

讓 Codex 修好外掛卡住的問題

能實現的效果

查日誌、看程序，定位瀏覽器或電腦控制連不上的原因。

複製給 Codex

請你作為 Codex 外掛故障排查助手，先檢查外掛狀態、日誌、後臺程序和鎖檔案，判斷為什麼 Browser 或 Computer Use 連不上；不要直接結束程序或重啟電腦，先說明風險和操作順序，等我確認。

X 經驗 bonitadreama 桌面操作
04客服跑腿

讓 Codex 幫你等客服、辦退款

能實現的效果

盯住客服視窗，整理上下文，先寫好回覆草稿。

複製給 Codex

請你作為客服跑腿助手，幫我盯住這個客服頁面，定時檢查是否輪到我；把要回復的話先寫成草稿，涉及傳送、退款、取消訂單或承諾條件時必須先讓我確認。

X 實測 jasper Computer Use

讓 Codex 幫你跑網頁流程

請你作為網頁流程跑腿助手，先只讀檢查這個網頁流程，理解每一步要做什麼；可以幫我填寫草稿和整理待辦，但涉及提交、付款、傳送、邀請、刪除或釋出時必須先讓我確認。

複製給 Codex

請你作為本地網頁 QA 助手，開啟我的本地頁面，圍繞這次 UI 改動完成真實點選測試；記錄 bug、截圖位置、復現步驟和建議修復順序。

08專案規則

讓 Codex 寫專案說明書

能實現的效果

複製給 Codex

請你作為專案規則整理助手，閱讀這個專案的 README、配置檔案和常用指令碼，生成一份適合 Codex 讀取的 AGENTS.md：寫清專案目標、執行方式、注意事項、驗收標準和不能擅自做的動作。

09Skill 工作流

給 Codex 裝上技能包

能實現的效果

讓常用流程自動觸發，少重複解釋。

複製給 Codex

請你作為 Skill 工作流助手，幫我判斷這個任務適合啟用哪些 Skill；優先考慮 brainstorming、writing-plans、TDD、code review 這類能降低返工的流程，並告訴我每個 Skill 具體幫我避免什麼問題。

10長任務

讓 Codex 跑完大任務

能實現的效果

分階段推進，只在關鍵決策時回來問你。

複製給 Codex

請你進入長任務模式，以這個目標為準持續推進；先寫完成標準和階段計劃，再執行最小可驗證步驟，遇到風險、許可權、費用或不確定決策時回來問我。

11程式碼審查

讓 Codex 提交前審一遍

檢查風險、缺失測試和可能的 bug。

請你作為程式碼審查助手，先閱讀這次改動的 diff，再按 bug 風險、使用者影響、缺失測試、可維護性排序列出問題；不要重寫程式碼，先給我審查結論。
